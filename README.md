# Ryan Joshy - Video Director Portfolio

A cinematic, award-winning portfolio website showcasing high-octane gaming montages and psychological edits. Built with vanilla HTML, CSS, and JavaScript featuring advanced GSAP animations and 3D Spline integration.

## 🎬 Features

- **Interactive 3D Hero** - Spline-powered 3D model with scroll-driven animations
- **Cinematic Preloader** - CSS 3D spinning cube with timed reveal
- **Stacking Card Gallery** - GSAP ScrollTrigger pinned carousel effect
- **Teal Peel Footer** - Split-complementary color transition with clip-path reveal
- **Magnetic Interactions** - Elastic button effects and custom cursor
- **Smooth Scrolling** - Lenis integration for premium feel
- **Dark Crimson Theme** - Monochromatic #3E1515 palette with atmospheric lighting

## 🛠 Tech Stack

- **HTML5** - Semantic structure
- **CSS3** - Modern Grid/Flexbox, Glassmorphism, 3D transforms
- **Vanilla JavaScript** - No frameworks
- **GSAP 3.12.5** - Animation engine
- **ScrollTrigger** - Scroll-driven animations
- **Lenis 1.0.29** - Smooth scrolling
- **Spline 3D** - Interactive 3D models

## 🚀 Getting Started

1. Clone the repository
```bash
git clone <your-repo-url>
cd PTF_web
```

2. Open `index.html` in a browser
   - Or use a local server (e.g., Live Server extension in VS Code)
